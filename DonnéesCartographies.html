Projet Cartographie

Présentation :

Le but du projet et de montrer l'évolution du Covid-19 en Europe.

Grâce à data.gouv.fr j'ai pu trouver les informations nécessaire pour avoir une table fusionné :Table CSV

La carte :

Avec les données présentent sur cette carte on peut répondre à quelques questions :

  • -A quelle niveau le Covid-19 a-t-il était présent sur le teritoire Européen ?
  • -Quels sont les pays les plus touchés (mort) par le Covid-19 ?

    • Voici le lien vers la carte : Carte

    La création et les informations de la carte :

    Afin de créer la carte il faut d'une part importer Folium avec : import Folium

    On entre ensuite les caractéristiques de base tel que : c= folium.Map(location=[51,10 ],zoom_start=5), j'ai choisi le centre de l'Europe comme location de départ.

    J'ai aussi créer des popup afin de mettre les infos à l'intérieur : folium.Marker([50.74599 , 9.766473],popup="text"

    Pour les graphiques j'ai eu beaucoup de mal, je n'ai pas réussi à mettre les données de ma table dans des graphiques, j'ai par la suite essayé de le faire à la main mais encore une fois j'ai manqué de savoir.

    J'ai quand même voulu rendre quelque chose et est donc opté pour une méthode moins belle et de moins bonne qualité : j'ai écrit une info pour chaque pays. Par exemple : folium.Marker([50.74599 , 9.766473],popup="En Allemagne, le 28 avril 2020, 6161 morts on été causé au total par le Covid-19").add_to(c)
